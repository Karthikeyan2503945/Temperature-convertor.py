## Features

-> Convert between:

  **Celsius → Fahrenheit / Kelvin
  **Fahrenheit → Celsius / Kelvin
  **Kelvin → Celsius / Fahrenheit

-> Automatically saves each conversion to conversions.txt
-> User-friendly text-based interface
-> Handles invalid input safely

## How It Works

The program:
1. Asks the user for a temperature value
2. Asks which scale to convert from (C/F/K)
3. Asks which scale to convert to (C/F/K)
4. Prints the result
5. Saves it in the format:
  ## 100 C -> 212.00 F
6. Asks the user if they want to perform another conversion
